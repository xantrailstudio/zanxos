# zanX OS (v8.2)

**zanX OS** is a high-performance, web-based operating system environment built to reimagine the cloud computing experience. This project was developed as a deep-dive into **Next.js**, **React**, and **Firebase** to master complex state management, windowing systems, and real-time cloud synchronization.

## 🚀 Overview
Experience a full operating system directly in your browser. zanX OS provides a fluid, desktop-like interface optimized for 60FPS performance, allowing for instant application switching and secure, persistent sessions.

## ✨ Key Features
- **Window Management:** Advanced system for handling multiple applications with a native desktop feel.
- **Cloud Integration:** Real-time data syncing across devices.
- **Authentication:** Secure user sessions powered by Firebase Authentication.
- **Performance:** Built for speed with optimized rendering for a lag-free experience.
- **Responsive Design:** Accessible from any device, bringing a "Cloud Computer" to your pocket or desktop.

## 🛠️ Tech Stack
- **Framework:** [Next.js](https://nextjs.org/) (React)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/)
- **Backend/Auth:** [Firebase](https://firebase.google.com/)
- **Deployment:** [Vercel](https://vercel.com/)

## 📂 Project Purpose
This project is a "learning milestone" created by the founder of Xantrail Studio. It serves as a practical exploration of:
- Building complex, scalable UI components in React.
- Implementing secure cloud-based authentication flows.
- Managing global state in a multi-window environment.
