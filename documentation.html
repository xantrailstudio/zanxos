Overview

zanX OS 8.2 is a complete reimagining of the ZanCore architecture, rebuilt for speed, security, and aesthetics. This release marks a significant milestone in web-based desktop experiences, introducing deep hardware-like integration using cloud-native technologies.

Key Features

macOS-style Lock Screen

A premium, blurred entry screen that securely separates your session. Features a beautiful dynamic clock and personalized user profile.

Cloud Theme Sync

Your wallpaper and profile settings follow you everywhere. Powered by Firebase, zanX OS synchronizes your appearance preferences across devices.

System Preloader

A high-performance initialization sequence that optimizes system resource loading before revealing the workspace.

Unified Settings

Manage your profile, change passwords, and customize themes from a glassy, centralized control center.

Global Error Guard

A robust error handling system that catches system-level exceptions and provides a safe "Critical Error" recovery interface.

Advanced Multi-tasking

A sophisticated window management system that handles complex app interactions with smooth, 60FPS animations.

Security & Privacy

zanX OS 8.2 introduces Persistent Security Layers. Unlike previous versions, your lock screen password is now encrypted and stored in your Firebase User Profile. The system automatically detects if a password is set and dynamically switches between "Instant Access" and "Secure Login" modes.


Authentication is handled through industry-standard Firebase Auth, supporting both Email/Password and Google OAuth providers.

Technical Architecture

Modern ES6+ JavaScript

Built with modular, asynchronous JavaScript for non-blocking performance.

Glass-morphism CSS

Utilizing backdrop-filter and modern CSS variables for a high-end visual aesthetic.

Firebase 10.7.0 SDK

Deep integration with Firestore (NoSQL) and Authentication for real-time cloud data sync.
